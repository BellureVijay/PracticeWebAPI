# ASP.NET Core Web API Practice Project

This is a backend practice project where I am implementing real-world concepts in ASP.NET Core instead of just building basic CRUD APIs.

As of **04-03-2026**, the project includes the following:

## What I Have Implemented

* Custom global exception handling middleware implemented in three different ways
* Minimal APIs
* Rate limiting using built-in ASP.NET Core rate limiter
* A controller to experiment with thread pool behavior and async execution
* External API calls using `IHttpClientFactory`
* Service-to-service communication simulating microservice interaction
* Retry and Circuit Breaker patterns for handling transient failures
* Background service that calls an external API every 20 seconds
* IMemoryCache
* Authentication using Hashing Mechanism(Register,Login, ForgotPassword,ResetPassword)

## Purpose

The goal of this repository is to practice and understand backend concepts deeply and prepare for building scalable, production-ready systems.

More enhancements will be added as I continue learning and experimenting.
